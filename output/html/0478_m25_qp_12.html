Q1a 7 marks AO1 Complete 1.1 Number Systems p.2▾

Complete the sentences about number systems. Use the items from the list. Some of the items in the list will not be used. You should only use an item once. A B C D E F G H W X Y Z 0 1 2 4 8 10 16 127 128 255 256 The binary number system is base [blank]. The smallest denary number that can be represented as an 8-bit binary number is [blank]. 1 mark for each correct item (in bold)

  • The smallest denary number that can be represented an 8-bit binary number is 0.[1]
  • The largest denary number that can be represented an 8-bit binary number is 255.[1]
  • The hexadecimal number system is base 16.[1]
  • Each hexadecimal digit is equivalent to 4 bits.[1]
  • The numbers 1 to 9 are used and the number 10 is represented by A.[1]
  • The hexadecimal number system continues up to the number 15, which is represented by F.[1]

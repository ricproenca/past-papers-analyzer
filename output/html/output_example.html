Q2 4 marks AO2 Explain 3.3 Data Storage p.4▾

Explain one benefit and one drawback of using solid state drives (SSDs) compared to hard disk drives (HDDs).

Marking scheme

1 mark for naming, 1 mark for explaining each

  • Benefit: faster read/write speeds / no moving parts[1]
  • Benefit: because data is accessed electronically not mechanically[1]
  • Drawback: more expensive per gigabyte than HDD[1]
  • Drawback: because flash memory cells are costly to manufacture[1]

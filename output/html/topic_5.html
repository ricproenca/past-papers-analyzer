Q4a 6 marks AO2 Identify 5.3 Cyber Security p.8 w25 · QP13 · Q4a▾

A cyber security company specialises in security solutions that help prevent hackers being successful. Identify two security solutions that can be provided to help prevent hackers being successful. Explain how each solution helps prevent hackers being successful.

Marking scheme

One mark for the security solution Two marks for a matching description

  • Strong/complex password[1]
  • Requires a mixture of letters, number and symbols[1]
  • Makes it very difficult for the hacker to guess[1]
  • Limiting number of attempts for password[1]
  • Cannot keep repeatedly guessing a password[1]
  • Stops a brute-force attack being successful[1]
  • Biometric device[1]
  • Requires the hacker to have biometric data to access the data[1]
  • It is very difficult to fake biometric data[1]
  • Two-step verification // two-factor authentication[1]
  • Requires the hacker to have additional data to access the data[1]
  • The data is sent to the users registered device/account that the hacker would also need[1]
  • Firewall[1]
  • Examines incoming data/traffic[1]
  • Can block any data that does not meet set criteria[1]
  • Anti-spyware[1]
  • Detects/removes any spyware present on the computer[1]
  • Hacker cannot use spyware to get data to discover passwords[1]

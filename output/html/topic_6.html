Marking scheme

Any five from:

  • Sensor continuously sends the digitised value / reading / distance to the microprocessor[1]
  • Microprocessor compares the data / signal to the stored data of a person and distance of 3m[1]
  • If the data / signal is less than (or equal to) a person and within 3m[1]
  • …a signal is sent to actuator to make the tractor stop / apply the brakes[1]
  • If the data/signal is greater than 3m no action is taken[1]
  • If stopped and data/signal is not a person and/or more than 3m a signal is sent to actuator to make the tractor start[1]
  • The whole process repeats until turned off[1]
